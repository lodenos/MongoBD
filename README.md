# MongoBD

Binding of MongoDB in Crystal

http://mongoc.org/libmongoc/1.17.1/api.html

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     mongodb:
       github: lodenos/mongodb
   ```

2. Run `shards install`
